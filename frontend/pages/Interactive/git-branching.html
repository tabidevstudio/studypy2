Commands Cheatsheet

  • git commit - Creates a new commit node
  • git branch <name> - Creates a new branch pointer
  • git checkout <name> - Switches to branch or commit
  • git merge <branch> - Merges changes into active branch
  • git checkout -b <name> - Creates and switches to branch
  • git log - Prints the commit history in console
  • clear - Clears the terminal screen
  • reset - Wipes the tree and restarts
