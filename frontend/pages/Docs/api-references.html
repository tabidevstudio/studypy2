API References

A curated directory of free APIs for CS/IT students. Each card includes a live "Try It" panel with a real code example you can run directly in your browser.

No Key Needed
GitHub API
Search repositories, view user profiles, list commits, and explore open source projects. No API key required for public endpoints — perfect for learning.
Docs Visit
Search a GitHub user — no key needed
const username = "torvalds"; const res = await fetch(`https://api.github.com/users/${username}`); const data = await res.json(); console.log(data.name, data.public_repos);
Free Demo Key
NASA APOD
Astronomy Picture of the Day. Get stunning space imagery with explanations written by professional astronomers. NASA provides a free demo key — no signup needed.
Docs Get Key
Today's Astronomy Picture — uses NASA's free DEMO_KEY
const KEY = "DEMO_KEY"; // NASA's free key — no signup const res = await fetch( `https://api.nasa.gov/planetary/apod?api_key=${KEY}` ); const data = await res.json(); console.log(data.title, data.url);
Stable
OpenWeatherMap
Access global weather data including current conditions, forecasts, and historical maps via simple JSON endpoints. Free tier allows 60 calls/minute — more than enough for projects.
Docs Get Key
Paste your free API key from openweathermap.org
const KEY = "YOUR_API_KEY"; const city = "Manila"; const res = await fetch( `https://api.openweathermap.org/data/2.5/weather` + `?q=${city}&appid=${KEY}&units=metric` ); const data = await res.json(); console.log(data.main.temp, data.weather[0].description);
Popular
NewsAPI
Search and retrieve live articles from over 80,000 news sources and blogs worldwide. Free developer tier for personal and learning projects. Great for building news aggregators.
Docs Get Key
Paste your free API key from newsapi.org
const KEY = "YOUR_API_KEY"; const topic = "technology"; const res = await fetch( `https://newsapi.org/v2/everything` + `?q=${topic}&apiKey=${KEY}&pageSize=3` ); const data = await res.json(); data.articles.forEach(a => console.log(a.title));
AI / LLM
OpenAI API
Access GPT-4o, DALL-E, Whisper, and Embeddings models. New accounts get free credits — enough for hundreds of test requests. The most widely used AI API in student projects.
Docs Get Key
Paste your OpenAI API key (starts with sk-)
const res = await fetch("https://api.openai.com/v1/chat/completions", { method: "POST", headers: { "Authorization": `Bearer YOUR_KEY`, "Content-Type": "application/json" }, body: JSON.stringify({ model: "gpt-4o-mini", messages: [{ role: "user", content: "Hello!" }] }) }); const data = await res.json(); console.log(data.choices[0].message.content);
No Key Needed
REST Countries
Get data about any country — population, capital, currencies, languages, flags, and borders. Completely free with no authentication required. Great for geography projects.
Docs Visit
Search any country — no key needed
const country = "Philippines"; const res = await fetch( `https://restcountries.com/v3.1/name/${country}` ); const [data] = await res.json(); console.log(data.name.common, data.population);
